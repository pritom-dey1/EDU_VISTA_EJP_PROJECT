# EDU Vista -- Course Selling Platform

A modern, fully responsive course-selling platform built with
**Next.js**, designed for students, instructors, and educational content
creators.\
Users can **register, log in, browse courses, purchase courses, and
instructors can add new courses**.\
Clean UI, smooth UX, secure structure, and fast performance.
